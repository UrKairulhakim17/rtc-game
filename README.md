# Web Game Project

Ini adalah projek permainan web mudah.

## Cara Pemasangan dan Menjalankan

Untuk memuat turun dan menjalankan permainan ini di komputer anda, sila ikuti langkah-langkah di bawah.

### Keperluan
- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.com/)

### Langkah-langkah

1.  **Clone repositori ini ke komputer anda:**
    ```bash
    git clone <URL_REPOSITORI_ANDA>
    ```
    *Gantikan `<URL_REPOSITORI_ANDA>` dengan URL sebenar repositori GitHub anda.*

2.  **Navigasi ke direktori projek:**
    ```bash
    cd test-2
    ```

3.  **Pasang semua kebergantungan (dependencies) yang diperlukan:**
    ```bash
    npm install
    ```

4.  **Mulakan pelayan web tempatan (local web server):**
    ```bash
    npm start
    ```

5.  **Buka permainan dalam pelayar (browser):**
    Selepas pelayan dimulakan, buka pelayar web anda dan layari alamat yang dipaparkan di terminal. Biasanya ia adalah:
    [http://127.0.0.1:8080](http://127.0.0.1:8080)

Selamat bermain!
